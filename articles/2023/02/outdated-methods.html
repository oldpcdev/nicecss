Why i use some “outdated” methods?

I know the base of the site looks like sites from 10-15 years ago, and that's on purpose.

The reason I use these methods is the ease and control I have over the elements.

The secret is that when you design in a professional image editor (GIMP, Photoshop, etc...) you can increase the detail capacity of your website to make it unique.

These details can be applied using images; Backgrounds, borders, icons and gradients.

Yes. I know some of these details can be added via pure CSS, but pure CSS has no soul as it's all generated by just a few lines of code. The method of images, however, does not, because it is composed of several pixels and different colors.

Of course I use some CSS3, like @font-face and box-sizing. The fonts are not that important but the box-sizing is crucial for the structure of the elements.
